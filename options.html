jQueryMobile - AlertBox

An "alert" box automator for jQueryMobile

AlertBox is a timed JavaScript "alert();" replacement, with automation, for jQM

jQM-AlertBox

General Use

AlertBox just uses a standard popup, without a corresponding button. You need only add the class "jqm-alert-box" to it, and AlertBox will take care of the rest

FAQ:

Can I have multiple alerts per site?
Of course - it was done on this site
Can I have multiple alerts per page?
No. (and I will never add this - there are ways to do it, none of them pretty - you are on your own)
Add what class name?
"jqm-alert-box"

Options

AlertBox supports setting a few extra options on the popup div:

data-alertbox-close-time
How long should the box be open? 5sec (5000ms) is default. This page overrode that to 2sec (2000ms)
data-alertbox-transition
Takes the place of data-transition on the popup link
data-alertbox-dismissable
Takes the place of data-dismissable on the popup link
data-alertbox-position-to
Takes the place of data-position-to, but probably "window" is a good idea since it doesn't use an anchor, eh?
